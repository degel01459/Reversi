git clone url
copia lo que hay en el enlace a la computadora

git add +nombre archivo
Para crear una estacion de trabajo antes de montarlo.
git add .

git commit -m "" (agregamos un comentario de las modificaciones)
es para subir una version con punto restauracion.

git push (todo lo que esta en la carpeta)
para finalmente montarlo se puede restaurar.

git pull (sincroniza todo lo que esta en la carpeta)
para actualizar cambios de carpeta sincronizada.

git pull --allow-unrelated-histories (saltar error de correlacion)

git config --global --edit (Configuracion del estado publico o privado)

git commit --amend --reset-author ()
